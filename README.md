# 🌱 Plant Disease Detection

This project detects plant diseases from leaf images using a deep learning model.

## 📌 Features
- Upload plant leaf images
- Detect plant diseases using AI
- Simple web interface using Streamlit

## 🛠 Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow

## 📂 Project Structure
app.py – Streamlit web application  
requirements.txt – Required Python libraries  
plant_disease_model_final.keras – Trained AI model  

## 🚀 How to Run the Project

1. Clone the repository

git clone https://github.com/Git-Aditi03/plant-disease-detection

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py
